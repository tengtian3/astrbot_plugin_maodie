# 耄耋来咯表情包插件

作者：腾天  
版本：1.1.0

## 插件介绍

一个有趣的AstrBot插件，当用户发送"哈个气"时，机器人会自动回复"耄耋来咯～"并附带随机表情包。支持表情包的管理和收藏功能。

## 功能特性

### 🎯 核心功能
- **自动触发**：发送包含"哈个气"的消息即可触发随机表情包
- **智能回复**：文字和表情包在同一条消息中发送
- **随机选择**：每次从表情包库中随机选择一张图片

### 📁 表情包管理
- **添加表情包**：通过命令添加新的表情包到收藏库
- **查看列表**：查看表情包统计信息和最近添加的图片
- **手动发送**：随时发送随机表情包
- **清理功能**：管理员可清理所有表情包

## 安装说明

1. 将插件文件夹 `astrbot_plugin_maodie` 放置到 AstrBot 的 `data/plugins/` 目录下
2. 重启 AstrBot 或重新加载插件
3. 插件会自动创建 `maodie_images` 文件夹用于存储表情包

## 使用指南

### 基础使用
用户：哈个气
机器人：耄耋来咯～ [随机表情包]

text

### 命令列表

| 命令 | 权限 | 功能说明 |
|------|------|----------|
| `添加表情包` | 所有用户 | 回复图片消息使用，将图片添加到表情包库 |
| `表情包列表` | 所有用户 | 查看表情包统计信息和最近添加的图片 |
| `随机表情包` | 所有用户 | 手动发送随机表情包 |
| `清理表情包` | 仅管理员 | 清理所有表情包文件 |

### 使用示例

**添加表情包：**
1. 发送一张图片
2. 回复该图片并输入 `添加表情包`

## 技术细节

- **兼容性**：支持 AstrBot 3.4.0 及以上版本
- **图片格式**：支持 JPG、PNG、GIF、BMP、WebP 等常见格式
- **存储路径**：表情包存储在插件目录内的 `maodie_images` 文件夹
- **文件命名**：自动生成唯一文件名避免冲突

## 注意事项

1. 首次使用时表情包库为空，需要先添加表情包
2. 建议使用清晰、有趣的表情包图片
3. 管理员可使用清理功能管理存储空间
4. 插件会自动创建必要的目录结构
